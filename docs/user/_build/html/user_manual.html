受講予約システム¶

受講予約方法¶

受講予約をしたい日付の時限をクリックします。 カレンダーの上にある月をクリックすることで、次の月や前の月に移動することができます。

  • 現在の月より前の月に移動することはできません。
  • 一つの時限で予約できるのは6名までです。
  • 数字が6になっている時限はそれ以上予約することができません。

受講予約画面が表示されるので、自分の学籍番号と氏名を入力します。 学籍番号は半角で、入力するようにしてください。

受講予約をキャンセルするときに、入力した学籍番号と、氏名が必要になりますので、間違えないように入力してください。

確認メッセージが表示されるので、OKをクリックすると受講予約は完了です。

キャンセル方法¶

予約した日付の時限をクリックします。

予約時に入力した自分の学籍番号と氏名を入力します。

[予約キャンセル]ボタンをクリックすると、確認画面が表示されます。

[OK]をクリックすると受講予約がキャンセルされます。 予約がないと表示された場合には、予約されていないか、学籍番号、氏名が予約時に入力したものと違う可能性があります。

キャンセルできなければ、別途サポートへご連絡ください。
